## Naming Notes

- Prefer SVG when available.
- Keep filenames lowercase with hyphens.
- Normalize display names in code where needed:
  - `RBC` -> `Royal Bank of Canada`
  - `Amazon.com` -> `Amazon`
  - `Bose Headphones` -> `Bose`
  - `OpenAI Tokens` -> `OpenAI`
  - `Shopify Checkout` -> `Shopify`
  - `Walmart.com` -> `Walmart`
